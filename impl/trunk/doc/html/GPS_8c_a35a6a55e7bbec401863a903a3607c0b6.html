int g_halGps_getData_i32 ( void  )

Parse Data from Uart into variables.

Author
Oliver Breuning ( olbrgs00 )
Date
2015/04/21

Reads data from the Hardware Driver and parses it in the specific important variables. Checks just the GPGGA and GPRMC Messages which are received from the GPS module. The return value gives back how many important Bytes are received.
