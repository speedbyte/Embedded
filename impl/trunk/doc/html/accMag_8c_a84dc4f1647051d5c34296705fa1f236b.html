Wrapper function to read byte stream to registers of LSM303D.

Author
Juergen Schmidt (juscgs00)
Date
2014/05/08

This function is a wrapper to the I2C Interface to read a byte stream to a series of registers of the Accelerometer/ Magnetometer (LSM303D). The first register address can be defined in the parameter list of the function. After reading the first byte of the start address, the register address will be incremented.

Parameters
[in]f_startRegisterAddr_ui8specifies the first register address to return
[in]f_readDestinationBuffer_pui8points to an array of unsigned 8bit integer elements. This array receives all read data bytes (output byte stream buffer).
[out]returnsa boolean value, indicating the occurrence of failures
0 .... indicates success (no errors)
1 .... indicates failed (some errors)
