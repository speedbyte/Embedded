Write-function for I2C0 Bus.

Author
Philipp Woditsch (phwogs00)
Date
2015/06/21

Function for writing a byte-array on the I2C-Bus. A pointer to the byte-array and the number of payload bytes to read are required. The function ensures to write the desired bytes on the bus, including a retransmission procedure.

Parameters
[in]f_lldI2c_i2cSlaveAddress_ui8is the address of the I2C Slave device
[in]f_i2cWriteBuffer_pui8is pointer to the byte-array to which the read bus data is written
[in]l_i2cNumOfDataBytes_ui32is the amount of bytes to be written
[out]returnsthe error state
0 ... success (no errors)
>=1 ... failed (some errors occured)
