static unsigned int l_updateSensorState_vd ( halAccmag_sensorSelect  f_sensor_en)
inlinestatic

Trigger a update of the current sensor state data.

Author
Juergen Schmidt (juscgs00)
Date
2014/05/05

This function accesses the accelereation and compass sensor via I2C to load a new measurement to the sensor state storage

Parameters
[in]f_sensor_enis a enum type that indicates, which sensor(s) shall be updated
[out]returnsa boolean value, indicating the occurrence of failures
0 .... indicates success (no errors)
1 .... indicates failed (some errors)
