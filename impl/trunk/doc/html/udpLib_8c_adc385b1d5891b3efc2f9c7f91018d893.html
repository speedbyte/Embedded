static signed short l_findSocket_i16 ( signed int  f_socketHandler_i32)
static

Finds the slot number of a given socket handler.

Author
Juergen Schmidt (juscgs00)
Date
2014/05/17

Since this module encapsulates the configuration data for each initialized socket, the slot number (=internal index number of the management buffer) is required. This function performs a lookup are returns the corresponding slot number.

Parameters
[in]f_socketHandler_i32specifies the socket handler number that shall be found
[out]returnsthe the internal slot number of the socket management buffer. If requested socket is not present, the value -1 will be returned
