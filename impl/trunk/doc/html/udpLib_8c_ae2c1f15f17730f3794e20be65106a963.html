Reads a packet of data via a given UDP-Socket.

Author
Juergen Schmidt (juscgs00)
Date
2014/05/17

This function reads a data packet via a given UDP-Socket. 

\param[in]  f_socketHandler_i32 specifies socket handler number used
            for the data transmission
\param[in]  f_sendBuffer_pui8 is a pointer to the receiving buffer
\param[in]  f_sendBufferSize_ui32 size (in bytes) of the receiving
            buffer
Parameters
[in]f_blocking_bldefines the blocking behavior of the UDP packet reading.
== 0 ... function blocks until packet received (default) != 0 ... function is non-blocking
[out]returnsthe number of payload bytes written to the receiving buffer. returns -1 in case of an error.
