Detailed Description

Container for time-stamped IMU state data.

Author
Juergen Schmidt (juscgs00)
Date
2014/05/20

Since the Udp-Lib is considered to handle a soft realtime connection, the IMU state data is considered to be time- stamped. This structure encapsulates a precise timestamp and IMU state vector.
