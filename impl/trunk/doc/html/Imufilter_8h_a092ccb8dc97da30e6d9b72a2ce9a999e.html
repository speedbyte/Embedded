void g_sigFil_readImuData_bl ( void  )

Trigger IMU reading.

Author
Oliver Breuning (olbrgs00)
Date
2015/05/31

Trigger IMU reading, correct the read value with the offset of the Gyroscope and convert magnitude from Tesla to µTesla. The Z-axis from Acceleration sensor and magnitude sensor gets 180° rotated.
