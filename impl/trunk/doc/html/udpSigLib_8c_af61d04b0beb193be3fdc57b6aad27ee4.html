Sends a packet of IMU data with the current unuix-timestamp via a given UDP-Socket.

Author
Juergen Schmidt (juscgs00)
Date
2014/06/01

This function sends a data packet with the given IMU state via a given UDP-Socket. The current timestamp is added to the payload to get some sort of realtime behavior.
This can be used to perform a Software-in -the-Loop setup or test specific filter algorithms in MATLAB.
It is recommended to open the neccessary UDP-socket with g_halMatlab_initSocket_i32() or g_halMatlab_initConnection_i32().

Parameters
[in]f_socketHandler_i32specifies socket handler number used for the data transmission
[in]f_imuState_stis a the IMU state packed into a custom struct type (type: halImu_orientationValues)
[out]returnsa boolean value, indicating the occurrence of failures
0 .... indicates success (no errors)
1 .... indicates failed (some errors)
