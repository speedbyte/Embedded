unsigned int g_halAccmag_initSensor_bl ( void  )

Initialize sensor.

Author
Juergen Schmidt (juscgs00)
Date
2014/05/05

Sends all commands on I2C to initialize the acc.- and magnetometer. It uses on big 8 byte config buffer to set all necessary bits. This way, a burst write to the control registers of LSM303D is performed to reduce the I2C bus load.

Parameters
[out]returnsa boolean value, indicating the occurrence of failures
0 .... indicates success (no errors)
1 .... indicates failed (some errors)
