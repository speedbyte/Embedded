halMatlab_rtSigAllStatePayload g_halMatlab_recvSigAllStates_bl ( int  f_socketHandler_i32)

Receives a packet of SIG data with the current via a given UDP-Socket.

Author
Juergen Schmidt (juscgs00)
Date
2014/06/01

This function receives a data packet with a synthetic SIG state via a given UDP-Socket. This can be used to perform a Software-in -the-Loop setup or test specific filter algorithms in MATLAB.
It is recommended to open the neccessary UDP-socket with g_halMatlab_initSocket_i32() or g_halMatlab_initConnection_i32().

Parameters
[in]f_socketHandler_i32specifies socket handler number used for the data transmission
[out]returnsthe received synthetic SIG state packed into a custom struct type with timestamp (type: halMatlab_rtSigPayload)
