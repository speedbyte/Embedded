Sends a packet of data via a given UDP-Socket.

Author
Juergen Schmidt (juscgs00)
Date
2014/05/17

This function sends a data packet via a given UDP-Socket. The payload is the given send buffer.

Parameters
[in]f_socketHandler_i32specifies socket handler number used for the data transmission
[in]f_sendBuffer_pui8is a pointer to the sending buffer containing the relevant payload data
[in]f_sendBufferSize_ui32specifies the number of payload bytes in the sending buffer
[out]returnsa boolean value, indicating the occurrence of failures
0 .... indicates success (no errors)
1 .... indicates failed (some errors)
