Configures a given UDP-Socket to communicate with a specific remote host.

Author
Juergen Schmidt (juscgs00)
Date
2014/05/17

This function sets the properties and binding of the given socket such that a proper communication with the destination address and destination port can be established.

Parameters
[in]f_socketHandler_i32specifies socket handler number used for the data transmission
[in]f_destIpv4_rg4ui8is a preallocated array of 4 bytes, each byte is one quadlet of a IPv4 address
[in]f_destPort_ui16determines the port number of the remote host that shall be used for the data transmission
[out]returnsa boolean value, indicating the occurrence of failures
0 .... indicates success (no errors)
1 .... indicates failed (some errors)
