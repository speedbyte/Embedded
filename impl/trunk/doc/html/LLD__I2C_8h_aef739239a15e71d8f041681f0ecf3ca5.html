Read-function for I2C-Bus.

Author
Philipp Woditsch (phwogs00)
Date
2015/05/04

Function for reading data from the I2C-Bus into a byte-array. A pointer to the byte-array and the number of payload bytes to read are required. The function ensures to write the desired bytes on the bus, including a retransmission procedure.

Parameters
[in]f_lldI2c_i2cSlaveAddress_ui8is the Address of the I2C Slave device in Hex
[in]f_lldI2c_i2cReadBuffer_pui8is a pointer to an array where the readed data should be stored
[in]f_i2cNumberOfReadingBytes_ui32isgives the amount of bytes to be readed
[out]returnsthe error state
0 ... success (no errors)
>=1 ... failed (some errors occured)
