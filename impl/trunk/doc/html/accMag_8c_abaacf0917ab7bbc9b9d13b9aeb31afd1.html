static unsigned char l_readI2cByte_ui8 ( unsigned char  f_registerAddr_ui8)
inlinestatic

Wrapper function to read a single byte register of LSM303D.

Author
Juergen Schmidt (juscgs00)
Date
2014/05/05

This function is a wrapper to the I2C Interface to read one single byte of the Accelerometer/Magnetometer (LSM303D). It eases the write and read procedure for reading one register byte of the sensor.

Parameters
[in]f_registerAddr_ui8specifies the register address to return
[out]returnsthe requested byte of I2C data
